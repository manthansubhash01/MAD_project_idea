## Features
- **Calendar View** – Visualize your schedule with a clean, interactive calendar layout for better planning.  
- **Smart Folder Structure** – Organize files, tasks, and notes using an intelligent and automated folder system.  
- **Notes Feature** – Create, edit, and manage quick notes with a smooth and minimal editor.  
- **Task Manager** – Track daily to-dos, manage priorities, and stay productive with an integrated task system.  
- **Progress Stats** – View analytics on completed tasks, productivity metrics, and overall progress.

## Description  
This project is a cross-platform productivity application designed to combine task management, note-taking, and calendar planning into one seamless workflow. It provides a powerful task system with reminders, prioritization options, and daily planning tools to help users stay organized and consistent. The note-taking module is built using a refined folder structure that ensures effortless creation, categorization, and retrieval of notes, even overcoming limitations of existing libraries.

A smart calendar view ties everything together, allowing users to visualize schedules and manage deadlines more effectively. To support long-term productivity, the app also includes detailed statistical dashboards that highlight trends, activity patterns, and progress insights. Overall, the system is built to streamline personal organization and give users a unified space to plan, document, and track their daily activities.

## Tech Stack

- **Backend:** Node.js + Express  
- **Database:** MongoDB
